## **📋 프로젝트 소개**

**SMART ANTS**는 SSAFY(삼성 청년 SW 아카데미) 프로젝트로 개발된 웹 서비스입니다.

사용자는 본 서비스를 통해 금융 관련 정보를 한 눈에, 쉽게 얻을 수 있습니다.

1. 주식 시장 정보

2. 예적금 상품 조회 및 추천

3. 금융 관련 기사

4. 금융 관련 영상(Youtube)

5. 금/은 현물 시세

6. 근처 은행/ATM/증권사 위치 조회
